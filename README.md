# 🦾 AI 虛擬機械手臂 × 資安防護 — 互動體驗

一個**不用安裝、不用登入、用手指點一點就能玩**的網頁，改編自實體課程
《AI 虛擬機械手臂與資安防護實務》。特別為**中高齡長輩與 3C 初學者**設計：
字大、按鈕大、全繁體中文、操作簡單。

## 🎮 四個攤位

| 攤位 | 你可以玩什麼 |
|------|------------|
| 🦾 **控制虛擬機械手臂** | 點按鈕或打中文（也能用說的🎤），指揮 3D 手臂分類積木、揮手、模擬故障；AI 即時用中文解說。還有「AI 安全攻防」小實驗，看懂什麼是「提示詞注入攻擊」。 |
| 🎣 **詐騙信件大考驗** | 一封一封判斷「真的 / 詐騙」，立刻看破綻解析，最後給你分數。 |
| 🔐 **密碼健檢機** | 輸入密碼即時測強度，提醒你是不是用了「最爛密碼」。 |
| 🛡️ **帶得走的防身術** | 五大資安習慣 ＋ 一張帶回家的表 ＋ 今天回去做三件事。 |

## ▶️ 怎麼玩

- **線上版**：直接打開 GitHub Pages 網址即可（手機、平板、電腦都可以）。
- **本機試玩**：把整個資料夾下載下來，用瀏覽器打開 `index.html` 即可。
  - 若「🎤 語音輸入」要能用，需要透過網址（https）開啟，並允許麥克風。

## 🛠️ 技術

純前端，零相依套件：`HTML + CSS + 原生 JavaScript`。
機械手臂用 SVG 繪製，搭配兩節手臂的**逆向運動學（IK）**做出流暢動畫。
可直接部署到 GitHub Pages 當靜態網站。
